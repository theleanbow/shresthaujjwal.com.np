## Overview

From August 2024 to January 2025, I completed a 5-month on-site apprenticeship at **INSPIRE** as a Solution Developer. The program focused on building real-world solutions to sustainability challenges using enterprise-grade tools and methodologies.

## What I Did

- Developed full-stack web applications using React.js, .NET, and relational databases
- Leveraged Azure cloud services for hosting, deployment, and integration
- Designed RESTful APIs and implemented Agile methodologies for project management
- Solved real-world challenges in sustainability, enhancing problem-solving and teamwork
- Participated in workshops on DevOps, Cloud Management, and Database Design with national and international mentors
- Applied IBM Garage Essentials and Enterprise Design Thinking to create impactful solutions
- Strengthened professional conduct and accountability in a collaborative environment

## Methodologies

The apprenticeship was grounded in **IBM Garage Essentials** and **Enterprise Design Thinking** — running empathy mapping sessions, defining user personas, and iterating through rapid prototyping before implementation. Agile sprints kept delivery on track.

## Skills Applied

ASP.NET · React.js · Azure · Azure DevOps · Git · Design Thinking · Empathy Mapping · UX · Agile
